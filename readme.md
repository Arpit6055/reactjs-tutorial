### Usage
First download this repo, move to this repo inside a terminal, do make sure you have **node**, **npm** installed

Then run the following command

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run dev

```
Visit http://localhost:1234/
